# 4Chat

A lightweight userscript that adds live auto-scrolling, a smart "Best Of"
filter, and more to 4chan. Great for stickies and generals. Compatible with
4chanX.

* 🔴 **Live Mode:** Makes the thread like Twitch chat, with you fixed at 
    the bottom and new posts coming in.  
* ⭐ **Best Of Mode:** Shows engaging posts based on who is replying to 
    whom.  
* ⚙️ **Customizable:** Click the gear icon to tweak update speeds, change 
    how many "Best Of" posts are shown, or hide buttons you don't use.  

### Installation

1. Install a userscript manager like **Violentmonkey** for your browser:
[Get Violentmonkey here](https://violentmonkey.github.io/#installation).  
2. Click here to install the script: [Install 4Chat
UserScript](https://raw.githubusercontent.com/chausies/chausies.github.io/refs/heads/main/4chat/4chat.user.js).  
3. Hit "Confirm" in the tab that opens up.  
4. Refresh any open threads, and you'll see the new menu in the top left.  

*This userscript was largely vibecoded with Gemini 3.1 Pro*
