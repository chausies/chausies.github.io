Send messages with public-key encryption!

Uses:

  • Enter a Password to get your public ID
  • Enter someone's ID and a Message to get an Encrypted Message only they can decrypt
  • Enter someone's Encrypted Message and your Password to get their original Message

No one can decrypt an Encrypted Message except for the person to whom the ID belongs.

Everything uses secure client-side javascript, so no information leaves your device. Your browser has already generated a secure random password for you by default, if you want to use it.

This website uses the Elliptic Curve Integrated Encryption Scheme (ECIES) algorithm with 256-bit security, Password-Based Key Derivation Function 2, and the Advanced Encryption Standard. This has security strength equivalent to 3072-bit RSA. For a full overview of this webapp and its security, see here. All the files for the webapp are guaranteed to be served from this GitHub repo, as can be independently verified here, by this CNAME lookup.

CAUTION: Make sure to use a strong password. If you use a bad password like "password", then someone could guess it and decrypt any messages.
